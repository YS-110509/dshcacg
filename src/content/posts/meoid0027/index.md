## 📋 违规类型分类

| 类型编号 | 违规类型 | 说明 |
|----------|----------|------|
| **A类** | 发布违规/不良信息 | 发布色情、暴力、恐怖、诈骗等内容 |
| **B类** | 传播谣言/虚假信息 | 散布不实信息，恶意引战 |
| **C类** | 人身攻击/不文明行为 | 辱骂、阴阳怪气、人身攻击等 |
| **D类** | 无关广告/刷屏 | 发布与二次元无关的广告、推销、刷屏内容 |
| **E类** | 涉政/敏感话题 | 讨论敏感政治话题、建政、引战性质内容 |
| **F类** | 诈骗/财产侵害 | 发布诈骗信息、诱导点击陌生链接等 |
| **G类** | 其他违规行为 | 其他影响群内秩序与氛围的行为 |
