## 📂 本站分类

社区里的内容会按不同板块整理，方便大家各取所好：

- **网站公告** — 官网功能迭代、维护公告、技术日志
- **官方公告** — 郑州大上海城官方活动公告
- **动漫讨论** — 番剧安利、观后感、角色分析，尽情交流你的心头好
- **漫画杂谈** — 连载吐槽、佳作安利、作者八卦，漫画爱好者的闲聊角
- **游戏专区** — 开黑组队、攻略心得、新游评测，从主机到手游都能聊
- **手工教程** — 同人周边制作、道具教程、痛物DIY，分享你的巧手成果
- **八卦交流** — 漫展趣事、社群日常、圈内见闻，没有什么不能唠的
- **Cosplay** — 正片分享、角色讨论、妆造心得，属于Coser的展示舞台
- **投稿展示** — 精选优质视频投稿内容，剪辑创作值得被看到

> 💡 板块分类会随着社群发展持续完善，如果你有好的建议，欢迎随时告诉我们！

## 📝 大上海萌资讯创作内容发布要求

为了让大家的内容能以最好的状态展示在官网上，同时也让投稿流程更加顺畅，请各位创作者仔细阅读以下要求：

### 🎨 投稿范围

我们欢迎一切与二次元相关的原创内容：

- **绘画作品** — 插画、漫画、同人图、表情包等手绘或板绘作品
- **Cosplay** — 正片、场照、妆造教程、角色还原心得
- **手工创作** — 道具制作、痛物DIY、黏土手办、周边改造等
- **文字创作** — 同人文、番剧安利、漫展Repo、游戏攻略、角色分析
- **摄影作品** — 漫展返图、宅物摄影、风景与二次元的融合创作
- **其他形式** — 只要你觉得与二次元相关且值得分享，都欢迎尝试！

### 📌 内容基本要求

| 要求 | 说明 |
|------|------|
| 🌟 **原创优先** | 投稿内容需为本人原创，严禁盗用、抄袭他人作品 |
| 📷 **图片质量** | 图片清晰、无明显水印，建议分辨率不低于1080P |
| ✍️ **文字规范** | 语句通顺、排版清晰、无大量错别字，建议附上适当配图 |
| 🔞 **内容尺度** | 遵守社群规范，禁止R18、血腥暴力等不适合公开展示的内容 |
| 🙏 **尊重角色** | 禁止恶意丑化、抹黑任何角色与作品 |

### 📬 投稿格式

请通过 **【大上海萌资讯创作服务】** 频道提交，需包含以下信息：

```
【作品标题】：（给你的创作起个名字）
【作品类型】：（绘画/Cosplay/手工/文字/摄影/其他）
【作者署名】：（你希望展示的名字）
【作品简介】：（简单说说创作灵感或想说的话，50-200字）
【作品文件】：（图片请发原图，文字请用可编辑格式）
【联系方式】：（选填，方便我们后续沟通）
```

### ⏱️ 审核与发布

- 投稿后我们会在 **3-5个工作日** 内完成审核
- 审核通过的内容将安排发布在官网对应板块
- 如需修改或不符合要求，我们会通过投稿渠道联系你
- 优秀作品将有机会在社群首页置顶展示或推送到官方宣传渠道

### 💡 温馨提示

- 投稿即视为授权大上海萌资讯在你的署名下公开展示该作品
- 投稿者保留作品的完整著作权
- 若后续希望撤下作品，请随时联系管理处理
